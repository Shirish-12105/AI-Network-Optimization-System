# AI Network Optimization System 🚀

## 📌 Overview
This project predicts network latency and optimizes bandwidth using Machine Learning and Deep Learning techniques.

## 🎯 Problem
Network performance depends on:
- Number of users
- Bandwidth
- Packet loss

This system predicts latency and suggests optimal bandwidth.

## ⚙️ Features
- Random Forest model for latency prediction
- Neural Network (Deep Learning) model
- Bandwidth optimization logic
- Synthetic data generation

## 🧠 Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow / Keras

## 📊 Output Example
- RF Predicted Latency: 164.2
- NN Predicted Latency: 143.38
- Optimal Bandwidth: 490
